## React API

We expose two components for the step indicator API, [StepIndicator and Step](https://github.com/wwnorton/design-system/tree/main/packages/react/src/components/StepIndicator).

```js
import { StepIndicator, Step } from '@wwnds/react';
```

<Callout title="Related concepts" border="left">

- HTML elements: [&lt;ol&gt;](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/ol), [&lt;li&gt;](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/li)
- ARIA property: [aria-current](https://www.w3.org/TR/wai-aria/#aria-current)

</Callout>

### StepIndicator

The `StepIndicator` component extends the `React.HTMLAttributes<HTMLOListElement>` interface and renders an `<ol>` element. It is composed using multiple `Step` components, which each render a `<li>` element. 

- When `connected`, the steps will render a connector line between each other, this is done using the pseudo-elements `::after` and `::before`.

```tsx live
<StepIndicator>
  <Step isCompleted>
    Order details
  </Step>
  <Step isCurrent>Payment</Step>
  <Step>Delivery</Step>
</StepIndicator>
```

### Step

The `Step` component extends the `React.HTMLAttributes<HTMLLIElement>` interface and renders an `<li>` element. It represents a single step.
