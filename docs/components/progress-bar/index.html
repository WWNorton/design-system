Progress Bar

A progress bar lets the user know that something is processing or progressing.

Anatomy#

  1. Track - a fixed width track for the progress indicator to travel along.
  2. Progress indicator - an animated bar that conveys processing or progress toward completion.
  3. Label - a description of what is progressing.

Usage#

Indeterminate vs. Determinate

The progress bar is said to be in "indeterminate" mode if it does not have a set value. This conveys to users that something is processing without conveying how close to being done it is.

When a progress bar does have a set value, it is said to be in "determinate" mode. This gives the user a better sense of how close it is to beng complete.

React API#

We expose one component for the progress bar API, the ProgressBar.

Related concepts
  • HTML element: <progress>
  • DOM interface: HTMLProgressElement
  • ARIA role: progressbar

ProgressBar#

The ProgressBar component extends the React.HTMLAttributes<HTMLDivElement> interface.
